让 ChatGPT 告诉我它可以完成的 50 件事:

💬 聊天与对话

  1. 闲聊
    示例： 和你谈谈今天的心情或讲个笑话。

  2. 情感支持
    示例： 你失恋了，我可以陪你聊聊、开导你。

  3. 角色扮演
    示例： 扮演哈利波特和你对话。

  4. 多语言聊天
    示例： 用法语和你练习日常对话。

  5. 模仿语气写作
    示例： 用“鲁迅”的语气来点评现代网红文化。

📝 写作与创作

  1. 写文章
    示例： 写一篇关于人工智能影响社会的议论文。

  2. 写小说/故事
    示例： 编一个关于时间旅行者的短篇小说。

  3. 写诗
    示例： 写一首七言绝句，主题是春天。

  4. 写剧本
    示例： 写一段办公室喜剧的短剧脚本。

  5. 撰写演讲稿
    示例： 为毕业典礼写一篇鼓舞人心的演讲稿。

📚 学习与教育

  1. 解释概念
    示例： 解释“相对论”是什么。

  2. 语法纠错
    示例： 帮我纠正这段英文：“She go to school every day。”

  3. 翻译语言
    示例： 把中文句子“我喜欢编程”翻译成西班牙语。

  4. 辅导功课
    示例： 讲解高中数学里的“函数图像”。

  5. 背单词练习
    示例： 帮我用“GRE核心词汇”出测验题。

👨‍💻 编程与技术

  1. 生成代码
    示例： 用 Python 写一个计算阶乘的函数。

  2. 代码解释
    示例： 解释这段 JavaScript 代码的作用。

  3. 代码调试建议
    示例： 为什么我的 Java 线程池任务执行失败？

  4. 算法教学
    示例： 教我如何用递归实现二叉树遍历。

  5. 系统设计建议
    示例： 给一个分布式聊天系统设计方案。

🎨 创意与设计

  1. Logo 点子
    示例： 给一家面包店想几个 logo 概念。

  2. UI 文案建议
    示例： 给“注册成功”的弹窗写几种提示语。

  3. 广告语创作
    示例： 为一个咖啡品牌写广告口号。

  4. 品牌命名
    示例： 为一个 AI 写作工具起名字。

  5. 短视频脚本写作
    示例： 为抖音写一个30秒的职场励志小剧本。

📈 工作与效率

  1. 制作计划表
    示例： 帮我安排一份三天复习考试的学习计划。

  2. 写简历
    示例： 帮我写一份后端开发工程师的英文简历。

  3. 写求职信
    示例： 给 Google 写一封求职信。

  4. 会议记录整理
    示例： 根据会议内容要点整理成笔记。

  5. Excel 公式编写
    示例： 写一个计算销售总额的 Excel 公式。

🌐 网络与数据

  1. 解析网站 HTML
    示例： 解释某个网页的 DOM 结构。

  2. 正则表达式生成
    示例： 生成一个匹配邮箱地址的正则。

  3. 网络安全基础教学
    示例： 教我什么是 SQL 注入。

  4. API 文档编写
    示例： 帮我写一个 RESTful API 的说明文档。

  5. 构建 Markdown 文档
    示例： 写一份 README.md 项目说明文档。

💼 商业与运营

  1. 商业计划书草案
    示例： 帮我写一个咖啡馆的商业计划。

  2. 市场分析
    示例： 分析智能手表行业的竞争格局。

  3. 用户调研问卷设计
    示例： 帮我设计一份 SaaS 产品用户满意度调查问卷。

  4. 预算计划制定
    示例： 帮我制定一份小型创业预算表。

  5. OKR 目标设定
    示例： 帮产品经理写一季度 OKR。

📊 数据分析与可视化

  1. 数据整理建议
    示例： 如何用 Pandas 处理缺失值。

  2. SQL 查询生成
    示例： 帮我写一个查询每月销售额的 SQL。

  3. 数据可视化建议
    示例： 这组数据适合用柱状图还是折线图？

  4. 生成图表代码
    示例： 用 Python 画一张饼图。

  5. 数据洞察分析
    示例： 从销售数据中找出高峰时间段。

🧠 其他能力

  1. 写信/道歉信
    示例： 帮我给朋友写一封道歉信。

  2. 制作谜语或脑筋急转弯
    示例： 编几个适合小学生的谜语。

  3. 帮助写结婚誓词
    示例： 帮我写一段感人的结婚誓言。

  4. 制定旅行计划
    示例： 帮我规划一次五天的日本关西自由行。

  5. 生成聊天机器人对话逻辑
    示例： 设计一个用于客服的聊天流程。
