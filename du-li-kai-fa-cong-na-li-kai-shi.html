独立开发从哪里开始？

产品变现的方式非常多样化，只要你的产品可以服务一小波人，一个垂直群体，就可以进行变现。可以通过虚拟权益、会员、卖课、付费社群、知识星球、小报童、竹白NewsLetter、爱发电、创作者收益、甚至微信转账等等。

  1. 找到一个需求和需求背后的人群（用户）

  2. 开发解决需求的产品并为产品设计买单方式

  3. 让你的产品触达用户

  4. 用户买单，你收钱

这个世界评判成功的标准太单一, 没赚到钱, 甚至亏本了, 那又怎么样, 尽情享受创造的快乐, 找寻编程的乐趣, 即使无名无利, Just for fun.

技术栈

技术栈一

  • 前端:
  • JS/TS with Vue3.0
  • BootStrap CSS framework
  • 后端:
  • Python Flask based Web App
  • Task process and scheduling by Cerely
  • 数据存储:
  • MySQL for business data
  • Redis for task queue
  • Minio for data file
  • InfluxDB for metrics
  • ElasticSearch for log search
  • 缓存及队列服务:
  • Redis
  • EMQX
