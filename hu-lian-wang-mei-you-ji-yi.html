互联网没有记忆

Posted on Sat 02 December 2023 in Journal

Abstract 互联网没有记忆

 Category    memory  
Status v1.0
Updated 2023-12-02

我一度曾经是微博的重度使用者，大约2004 年就开始在新浪上写博客，发微博，记录下生活中发生的事，记录自己对生活，工作以及人生的感悟。

如今我的新浪博客和微博除了我自己，别人都看不到了，前者是很长时间没更新，后者是被禁言了。

刚开始还有些沮丧，后来看到越来越多自己关注的人都主动或者被动地离开了，自己也就释然了，自己在自己的个人站点以及 github 上维护一个静态博客，用提交代码的方式来更新博客。虽然还是不能畅所欲言，起码不担心管理员随意封禁我的发言，因为我自己就是管理员.

可是我发现在缄默了许久之后，我似乎失去了表达的欲望，感觉变得迟钝了，即使重新开了一个微博小号，也没办法象以前那样踊跃的发言，随时随地想到什么就发条微博。部分原因是我担心微博再次被封，另一部分是心态有点丧，没有什么想说的。

原来我的个人博客基本上是每周更新，主要是写一些技术方面的笔记和感悟，最近有一个多月没更新了，原因还是心绪不佳，更换工作之后迟迟没有找到状态。

昨天听了一期“津津乐道”的播客，讨论的是滴滴宕机的事，其中聊到了天涯论坛的事，天涯上那么多的精华帖子都已经消失不见了，天涯论坛倒了，那么多的帖子也不找不着了。

我深有感触，互联网是没有记忆的，而人是有记忆的，不过人又是善忘的，很多往事我都已经记不得了，很多生命的过客在记忆中都已经模糊了。

从今天开始，我想多点时间记录生活，也把过去写的东西摘录一些我的博客里，互联网没有记忆，而我希望记住生活中所经历的喜怒哀乐
